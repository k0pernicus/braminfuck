# braminfuck
Braminfuck, a brainfuck interpreter in Nim.

# How it works?

```nimble build```

If you want to interpret a 'live' brainfuck program:  
```echo "++++++++[>++++[>++>+++>+++>+<<<<-]>+>+>->>+[<]<-]>>.>---.+++++++..+++.>>.<-.<.+++.------.--------.>>+.>++." | ./braminfuck```

If you want to interpret a brainfuck program:  
```./braminfuck myprogram.b```
